How should the "new / edit bakery" screen feel?

This is used constantly — onboarding a new bakery, tweaking a polygon, adding a depot. Pick a layout.

A. Single-page editor — sidebar form + live map

Bakery info and depot list on the left, full-height map on the right. Polygons and depots drawn on the same map, everything edited in place. One "Save" at the top-right. Works for both creating and editing.

B. Multi-step wizard (create only) + simple edit screen

C. Map-first, full-bleed — floating panels

Map takes the entire viewport. Bakery info and depot list live in a small floating panel top-left; draw toolbar floats top-right; save/cancel float bottom-right. Same screen for create/edit.
